# bash-scripts

A collection of bash shell scripts for RHCSA certification practice and general system administration tasks.

## Overview

This repository contains practical bash scripts developed for:
- **RHCSA exam preparation** - covering core Linux system administration topics
- **Day-to-day operations** - automating common sysadmin workflows

Each script is designed to be standalone, readable, and ready to use.

## Usage
```bash
# Clone the repository
git clone https://github.com/rzkw/bash-scripts.git
cd bash-scripts

# Make any script executable
chmod +x script-name.sh

# Run it
./script-name.sh
```

## Script Categories

**System Administration**
- User and group management
- File permissions and ownership*
- Process monitoring*

**RHCSA Topics**
- Storage and filesystems*
- Network configuration*
- Service management*
- Security and SELinux*

**Utilities**
- Log analysis
- Backup automation
- System health checks

(* = work in progress!)

## Prerequisites

- Bash 4.0 or higher
- Standard GNU utilities (most Linux distros)
- Root/sudo access for some scripts

## Notes for Beginners

Don't know bash scripting yet? No worries:
- Each script includes inline comments explaining the logic
- Start with simpler scripts to understand the patterns
- Most scripts use common commands you can Google
- Feel free to modify them for your own environment
